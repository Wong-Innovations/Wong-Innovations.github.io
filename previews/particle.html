Particle

I liked the idea of mutually assured destruction between matter/anti-matter collisions, so much so I created this canvas game based upon it! The full code can be viewed here.


The JavaScript backing the game can be broken down into three main components: an object-oriented Entity Component System to track each particle position and collision detection, a 'Render' class for drawing text and circles to the canvas, and an animation loop (serving as the 'main' function of the program).
