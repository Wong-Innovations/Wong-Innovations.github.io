Mafia

My friends and I love social party games like Mafia, and while game developement does usually interest me, I knew it would be a chance to learn react-firebase! The full code can be viewed here.


The React project uses the Material UI design library, and react-firebase to manage the state of the users. Users can login or create an account and they are placed into a lobby, when at least 5 people are in the lobby and "readied up" simultaneously a games is started. In the game users have buttons to vote and preform other role-specific actions, as well as a chat functionality to interact with each other.
