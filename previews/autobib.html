AutoBib

My first unguided project in React, my friends and I were upset with how many citation generator cites are plastered in adds and require paid subscriptions. So we began work on our own AutoBib cite, the full code can be viewed here.


The React project uses the Material UI design library, we have implemented custom citation classes (MLA8, APA, ...) which take the form data as paramaters and return formatted text to be displayed in the "paper" preview. Lastly we used API's like the CORE documents database to create a document search feature!
